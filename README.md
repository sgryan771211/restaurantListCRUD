# Restaurant list
Restaurant list 是一份美食口袋名單，收錄了各個餐廳的資料

---

## Install
開啟你的終端機
1. 輸入 git clone https://github.com/sgryan771211/restaurantListCRUD.git
2. 輸入 cd restaurantListCRUD
3. 輸入 npm install
4. 輸入 node app.js
5. 在瀏覽器網址上輸入 localhost:3000
6. 開始使用清單
---

## Features
1. 在首頁可以看到所有餐廳與它們的簡單資料
2. 可以再點進去看餐廳的詳細資訊
3. 可以透過搜尋餐廳名稱來找到特定的餐廳
4. 可以新增、修改及刪除餐廳資料
